# Proactive Agent

## 适用场景

当用户需要 AI 助手能够主动预测需求并持续改进时，使用该技能。

## 使用步骤

1. 配置工作缓冲区。
2. 实现自主定时任务。
3. 使用 WAL 协议记录操作。
4. 持续改进 AI 助手的能力。

## 示例

```bash
# 实现自主定时任务
python scripts/proactive_task.py
```
