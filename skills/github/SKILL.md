# GitHub

## 适用场景

当用户需要操作 GitHub 仓库时，使用该技能帮助用户完成相应的操作。

## 使用步骤

1. 配置 GitHub CLI (gh)。
2. 使用 gh 命令操作 GitHub 仓库。
3. 提供操作的详细信息和步骤。

## 示例

```bash
gh repo view shenyizhou/openclaw-workspace
```
